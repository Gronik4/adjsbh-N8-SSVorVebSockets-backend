 ___________________ Переменные для сервера:
 ***** При получении данных: ***
 "start" - входит любой user. Тип - строка.
 parsMessClient.entre - входит регистрированный user. Тип - строка.
 parsMessClient.registered - user пытается регистрироваться. Тип - строка.
 parsMessClient.mess - Получаем очередное сообщение. Тип - строка.
 parsMessClient.exit - Получаем очередного вышедшего из чата участника. Тип - строка.
 
 ****** При отправке данных: ***
 send(onlines: ....) - Отправляем список участников onlines из файла. Тип - buffer.
 send(onlin: ....) - Отправляем всем очередного участника onlin. Тип - строка.
 send(messages: ...) - Отправляем список сообщений из файла. Тип - buffer.
 send(mess: ...)  - Отправляем всем очередное сообщение участника. Тип - строка.
 send(registered: ...) - Отправляем в случае удачной регистрации участника. Тип - строка.
 send(exit: ...) - Отправляем всем вышедшего участника. Тип - строка.

 Это пробное сообщение просто для проверки, что работает.
